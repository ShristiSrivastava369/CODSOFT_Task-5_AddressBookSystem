Contact Management System

Website Buttons

Create New Contact

Create a new contact using this button. It will lead to a page where you can fill out a form and submit it. The contact will be saved in the database and displayed in a table.

Delete Contact

Use this button to directly remove a contact. It will be deleted from the database as well.

Update Contact

Click this button to update different fields of a contact. The changes will be updated in the database.

Search Contacts

Search for contacts using their name, phone number, email, cities, etc. The search is case-sensitive and will display all contacts that match the exact search criteria.
